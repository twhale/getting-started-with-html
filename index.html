Stage 0 Notes – Getting Started with HTML

HTML documents are a key component of internet communications, as it is the language that is passed between the user, the user’s computer, the internet, and the web servers.

HTML = Hypertext Markup Language. HTML translates to text, markup (tags for arrangement), references (images and documents) and links to other locations.

Tags are used to mark the beginning and ending locations for special formatting. Start and end tags are enclosed in < > 's.

There are 2 types of HTML tags – inline and block. Inline tags are used to format a single line of text, whereas block tags are used to define formatting for a section of text, like a paragraph.
Image an invisible line drawn around a section where block tags control the style of that entire area.

Examples of some Inline tags:
a - anchor tag; use to define file/locations
b - makes text bold
br - line break tag (this is a void tag; it does not require an end tag)
em - italicizes text

Example of a block tag:
p - paragraph tag

Attributes are used with tags to define files and file locations.
Use attribute href with an anchor tag to define a link to a web page, or src with the img tag to define a location of an image.
